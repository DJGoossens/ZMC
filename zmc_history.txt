! Aug  05  2009: First thing is to make writecrystal only work if you ask it to.              !
! As for .diffuse file, it should have a header with lots of info, plus a body with the       !
! data.  Now, header will need simsize.  Max number of possible atoms sites in cell and       !
! max number of occupied sites in cell.  These are for array allocations.  Then we need an    !
! array to translate.  Call it correspond(a,b,c,100)          For example, if we have a       !
! system in which we have 100 possible sites in the unit cell, but only 25 are occupied       !
! at a time, then we would put these numbers in the header.  Further, we would have an        !
! array of size a,b,c,100 whose values are the indices of the arrays with the labels and      !
! x,y,z coords in.  Then we would have other arrays x(a,b,c,25) etc which we would look       !
! at.  If we get passed in i,j,k,75, then we go to correspond().  If that cell is a zero,     !
! then atom site 75 is unoccupied in cell i,j,k and react accordingly.  If it is a number,    !
! say N, then x = x(i,j,k,N) etc and pass these values back to DIFFUSE.  Simple!              !
! This suggests that what we need to do is (1) Work out number of possible sites in a cell.   !
! (2) Work out the maximum number of sites that can be occupied in a cell.                    !
! (3) Write these things and simsize() to the output file.                                    !
! (4) Loopover the POSSIBLE sites in the cell (by looping over locations then possible        !
!     z-matrix then possible orientations) and increment a counter n for each non-dummy atom. !
!     Before doing this, set correspond() to zeroes; then, when on a site which IS occupied   !
!     increment a second counter, m, and set correspond(i,j,k,n) = m.  Then set               !
!     x(i,j,k,m), y, z and label() as well, so that they can be accessed by using             !
!     x(i,j,k,correspond(i,j,k,n)) for example.  Then, when doing readat() it is simply       !
!     that if correspond(i,j,k,isite)=0, it is unoccupied.  If .ne.0, then get label and      !
!     compare to type passed in.  If agreement, set the flag, set x,y,z and get out. Easy!    !
